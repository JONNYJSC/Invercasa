# Invercasa
***
## Leer Indicaciones
```
Hay un resplado del esquema de base de datos llamado InvercasaScritp.

Para configurar la conexion a la base de datos hay un archivo en el proyecto.web 
este se llama appsettings.development.json propiedad DefaultConnection

Este proyecto corre en DotNet 6 y hace uso de MSSQL Server
```
